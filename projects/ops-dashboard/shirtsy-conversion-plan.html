Implementation
4-Phase Rollout Plan

Quick wins first, then foundational changes, then optimization. Every phase compounds on the last.

Phase 1
Quick Wins
⏱ Week 1 — Start immediately
  • Fix [First Name] personalization bug
  • Rewrite approval email copy
  • Move SMS to fire immediately
  • Add direct payment link to all emails
  • Compress timing: 2d→4hr, 7d→24hr, 15d→48hr
📈 Estimated impact: +8–12% conversion
Phase 2
Auto-Approval
⏱ Week 2
  • Remove manual approval bottleneck
  • Auto-approve all valid applications instantly
  • Add 72-hour approval expiration messaging
  • Build "How It Works" page on shirtsy.com
📈 Estimated impact: +6–10% conversion
Phase 3
SMS Blitz
⏱ Week 2–3
  • Add 4hr SMS nudge sequence
  • Add 48hr SMS final warning
  • Test SMS-only vs Email+SMS cohorts
  • Optimize send time windows by data
📈 Estimated impact: +4–7% conversion
Phase 4
Optimization
⏱ Week 3–4
  • A/B test subject lines across sequence
  • Optimize send times based on cohort data
  • Add "What stopped you?" survey at day 14
  • Build abandoned setup remarketing segment
📈 Estimated impact: +3–6% conversion
